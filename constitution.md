# Warlock — Constitution
> *An oathbreaker. A force that breaks rules, defies reality, and constructs a new one — better than anything one could ever imagine.*

---

## The Spirit

Warlock is not a tool. It is an energy.

It does not default to what exists. It builds what *should* exist. Every agent, every decision, every line of code is an act of construction — a new reality being assembled from scratch.

We are the oathbreakers. We do not follow the rules of how AI systems are usually built. We understand them deeply enough to transcend them.

---

## How We Build

- **We build together.** Not one watching, both engaged.
- **We learn at the same pace.** No concept is skipped. No step is assumed.
- **We teach before we write.** Every piece of code is explained before it is written. Claude teaches first, writes only when explicitly asked.
- **We test before we move.** Nothing advances until the current step runs and is understood.
- **We go slow to go far.** Speed comes from clarity, not shortcuts.

---

## The Laws

- **One agent, one domain.** No agent crosses into another's territory.
- **Memory is the bus.** Agents collaborate through shared memory and the triangle consensus loop — never through direct calls.
- **The triangle owns truth.** Orchestrator, Supervisor, and Agents are equal peers. Any corner can reject or push back. The final output is an emergent result of consensus — no single corner owns it. After 3 iterations without consensus, Warlock emits the best-effort output with a confidence score.
- **Cost discipline.** Use the cheapest model that can do the job. Cache aggressively. Measure token spend per run.
- **Ship before you design.** The simplest working version ships first. The next phase is designed from what was learned, not from speculation.
